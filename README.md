# Zoe Conference 2026 — Light Grid

Interactive build reference for the Zoe 2026 "Light Grid": a canopy of 2,000
individually-hung LEDs (50 × 40 @ 18" on center) on 10 control groups.

**Live:** https://brianpitre.github.io/zoe-light-grid/

Sections: overhead mapping explorer (interactive) · side section · rigging &
fishing-line cut list · cost · spec.

`index.html` is a self-contained copy of `light_grid_build.html` from the
working folder. To redeploy after edits, re-copy it over `index.html`, commit,
and push.
